# Week 1 Assignment - Basic Data Exploration and Cleaning using Pandas

## Objective
The objective of this assignment is to learn Python basics and perform basic data exploration and cleaning using Pandas.

## Tasks Completed
The following tasks were performed in this assignment:

1. Loaded the CSV dataset using Pandas
2. Explored dataset structure using:
   - head()
   - shape
   - info()
   - describe()
3. Identified and handled missing values using:
   - fillna()
   - dropna()
4. Performed basic data operations:
   - column selection
   - row filtering
5. Checked and removed duplicate records
6. Created a derived column (`total_amount`)
7. Saved the cleaned dataset into a new CSV file

## Files Included
- `Week1_assignment.ipynb` : Jupyter notebook containing code
- `Combined_dataset.csv` : original dataset
- `cleaned_shopping_dataset.csv` : cleaned dataset
- `README.md` : assignment explanation

## Key Learnings
Through this assignment, I learned:
- How to load and explore datasets using Pandas
- How to handle missing values
- How to filter and analyze data
- How to remove duplicates
- How to create derived columns
- How to save cleaned data into CSV format

## Conclusion
This assignment helped improve my understanding of data exploration and cleaning techniques using Pandas. It also helped me understand how real-world datasets contain missing values, duplicates, and require preprocessing before analysis.
